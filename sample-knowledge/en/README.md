# Sample Knowledge Base (English)

A small test dataset for trying the RAG knowledge base in Open WebUI. The documents
are invented but internally consistent: the answers exist only in these files, so
you can clearly tell whether the system is really answering from the documents.

## Upload these four documents into a knowledge collection

- `it-policy.md` — the school's IT policy (Wi-Fi, passwords, printing, servers)
- `fire-safety-policy.md` — fire safety and evacuation
- `product-datasheet-packmule-x2.pdf` — a cargo bike's technical datasheet
- `excursion-hannibal-colliery.pdf` — an information sheet for a school trip

Two Markdown files and two PDFs, four different topics, so you can see how
retrieval finds the right source.

## Do not upload

- `question-sheet.md` — the questions to try, with expected answers.
- `README.md` — this file.

Upload the four documents, then work through `question-sheet.md`.
